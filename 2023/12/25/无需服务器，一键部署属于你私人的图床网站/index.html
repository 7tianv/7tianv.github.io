无需服务器，一键部署属于你的私人图床

作为一个博主，写文章必不可少的需要用到大量的图片，因此就需要用到图床。而把图片放在别人网站上，难免有点不靠谱。因此，自建图床的需求就来了，而且也可以分享图片给其他人，这不就非常Nice吗

开始的准备：vercel账号，github账号，域名

1.github账号注册

有GitHub账号的小伙伴可以跳过这一步

首先在 GitHub.com 注册一个账号，点击右上角的 Sing up即可注册

然后依次填入你的邮箱地址(国内外的邮箱都可以)，密码，用户名，y，然后点击验证，此时邮箱里会受到验证码，填入即可完成注册

2.vercel账号注册

有Vercel账号的小伙伴也可以跳过这步

方法同github注册一样，点击右上角的 Sing up

这里选择 Hoppy ，个人注册

这里任选一个注册即可，博主选择的是最下方的Email邮箱注册

最后在你的邮箱里点击链接验证即可完成注册

3.图床搭建

这里使用的是Github大佬开源的程序，https://github.com/x-dr/telegraph-Image ，点击Github仓库地址，找到 利用vercel部署(vercel分支) ，点击下方蓝色的 Deploy

点击后会自动跳转到vercel网站，点击 continue with Github 连接到github仓库，连接完成后会显示github用户名

在右边输入你想设置的仓库名称，点击右下方的 Create 按钮即可完成创建，第一次创建失败，换个名称再次创建即可成功

如图，显示这个界面说明创建成功了，点击右上角的 Continue to Dashboard 转到控制面板

图中红框的域名就是图床域名，不过由于vercel分配的域名在国内无法被dns解析到，因此需要添加我们自己的域名使用，也方便好记，点击右上角的 Domains

在方框内填入你想解析的域名，然后点击Add

会出现Name和Value，复制value的值

到你的域名购买商处进行解析，类型选择Cname，名称填入Name值，内容填入value值，因为vercel的cdn国内速度还是挺快的，因此不需要开启cf的cdn，点击保存

返回vercel，会自动刷新，不需要操作，证书也会自动申请，点击解析好的域名即可访问图床

属于你的专属图床就搭建好了，开始愉快的使用吧

如果你没有域名，可以在电脑hosts文件添加vercel的解析进行访问，教程请自行百度。更多配置请参考Github大佬的教程
