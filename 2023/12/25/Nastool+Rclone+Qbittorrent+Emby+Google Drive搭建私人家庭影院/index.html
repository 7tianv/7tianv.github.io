Nastool+Rclone+Qbittorrent+Emby+Google Drive搭建私人家庭影院

Nastool+Rclone+Qbittorrent+Emby+Google Drive搭建私人家庭影院

谁不想在结束忙碌的一天，舒舒服服的躺在床上观影呢，但想找一部想看的影视，需要辛辛苦苦的在网上找资源，然后下载，打开一看各种删减，广告铺面而来。就这样，观影的兴致都没有了，因此本期教程就让你一键观看自己喜欢的影视

本期教程面向有一定能力部署，如果你是小白，请自行决定是否操作

开始前的准备：

vps (推荐最低2c4g的配置，存储100G以上)

PT站账号 （推荐有，没有也可以）

大存储网盘 （推荐Google Drive，上传不限速）

教程开始前默认你已经在服务器上安装rclone并配置好了网盘，rclone挂载的网盘名称必须为 NASTOOL !!!

1.映射网盘到本地为本地磁盘

这里博主将网盘映射到 /home/NASTOOL 作为本地磁盘，在/etc/systemd/system下创建rclone.service文件，将下方的配置粘贴进去，然后保存

配置中NASTOOL为你使用rclone挂载网盘的名称，一直要为 NASTOOL ，只有原因后面在说明， /home/NASTOOL 是你需要挂载网盘到本地的路径，如果需要修改为你自己的路径，请将有关 /home/NASTOOL 的全部修改，否则会报错，修改完成后启动挂载

上面的命令依次执行，查看状态显示绿色的 running ，使用 df -h 命令查看 /home/NASTOOL 的空间大小，如果和你网盘空间大小一致，说明成功挂载了，finalshell左边可以直接看到路径和大小

2.安装nastool

由于nastool作者只提供了docker容器安装，因此我们需要事先安装好docker，本文不提供docker安装教程，请参考这位大佬的安装教程：Debian 12 / Ubuntu 22.04 安装 Docker 以及 Docker Compose 教程

然后将下方的命令复制粘贴到ssh上，按照自己的需求进行更改

粘贴后回车自动下载部署，然后访问 ip:3000 进入nastool面板

进入这个页面代表nastool安装成功了，默认登录账号 admin 密码 password ，进入后或提示你添加TMDB API和设置登录密码， TMDB API申请请参考这篇文章：TMDBAPI申请攻略

申请完之后将api填入，点击保存即可

3.安装Qbittorrent下载器

NASTOOL安装之后，就需要进行下载器的安装了，创建 /opt/qbittorrent-nox 目录，然后进入这个目录，将qbittorrent下载到这个目录，依次执行下方的命令即可

然后在 /etc/systemd/system 目录下创建 qbittorrent-nox.service 文件，然后将下方的配置粘贴进去

然后依次输入下方的命令启动qbittorrent

显示绿色的 running 说明创建成功了，访问 ip:8080 即可访问面板，默认登陆账号 admin ，密码 adminadmin ，请及时更改密码

4.安装Emby

访问 https://emby.media/linux-server.html ，选择自己的系统版本进行下载到服务器上，然后输入对应的命令进行安装

然后访问 ip:8096 ，即可登陆到emby服务端，进行初始化设置，然后进入设置-媒体库，点击新媒体库，将你网盘的映射路径添加进来

点击API密钥，添加新API密钥，将密钥复制下来，只需要复制图中红框内的部分

5.设置NASTOOL下载影视，自动刮削并上传到网盘

进入nastool，找到媒体服务器，点击Emby

填入你的Emby ip地址 端口号和api密钥，如果你和博主将emby安装在同一台服务器上IP地址可以填写 127.0.0.1 ，如果不是请按你实际的IP填写，点击测试，提示测试成功，点击确定保存即可

然后点击下载器，在里面填入你的ip地址，端口号，账户和密码，点击测试，确认成功之后保存即可

点击右上方的下载目录，填入你的下载路径即可，下载路径要和你的qbittorrent下载路径一致，博主qb的下载路径为 /Downloads ，可以创建子文件夹

找到 目录同步 ，新增同步目录，源目录里填入刚才创建的qb下载目录，目的目录直接填入你想在网盘创建的文件夹名称，未识别目录可不填，同步方式选择 Rclone移动 ，前面rclone名称如果不是NASTOOL，这一步同步到网盘会报错，导致无法将文件移动到网盘里

最后找到索引器，选择内建索引，如果你有PT站账号，可以添加并选择你的PT站点，如果没有，可以选在下方的公开站点

如果你有电报账号，可以申请电报机器人，将api填入，接受nastool的消息通知，也支持其他通知，请自行探索

6.测试

在探索里面找一部想看的电影，点击进去，再点击搜索资源。如果有会显示资源信息

然后点击右边的下载按钮，保存目录选择你对应的下载目录

推送到qb下载，成功会返回提示，同时电报机器人也同步提示下载信息

在qb里面也能看见文件正在下载

下载完成并同步到网盘后会返回已入库的消息

在emby里也能看见影视同步到库里了，点击即可播放

由于网页的限制，推荐使用客户端进行播放获得更好的效果，下方的网盘里有各平台的客户端，请自行下载安装使用

由于emby需要购买会员，如果是个人使用，则不需要在意，如何共享给其他人观看，请自行购买会员或寻找破解版客户端

这样，私人家庭影视库就搭建完成了，只需一键即可全自动完成影视观看
